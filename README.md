## 🔧 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Subh-29/Cohort.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Cohort
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
